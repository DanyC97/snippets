7.1. Stackoverflows¶

http://stackoverflow.com/questions/61212/how-to-remove-local-untracked-files-from-the-current-git-branch

# do a dry run
git clean -n

# do interactive (to be on the save side)
git clean -i

# include directory
git clean -n -d

# if happy with dry run, actually do clean
git clean -f -d
